**Device Welcome App**



A reference application for BaseOS and Hardware Abstraction Layer (HAL), demonstrating unified access to industrial communication, sensing, positioning, and connectivity services.



**Overview**

The Device Welcome App is a demonstration application running on BaseOS with HAL (Hardware Abstraction Layer).

The purpose of the application is to provide developers, integrators, and customers with a simple and intuitive entry point for exploring device capabilities and validating hardware interfaces.

The application abstracts hardware complexity and presents all available interfaces through a modern user-friendly dashboard.



**Currently supported devices**

✅ TDC-X

Future support planned for additional BaseOS compatible platforms.
